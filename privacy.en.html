Overview

This is our policy regarding how the mobile application Mamababa, built by Gene Lee, collects, uses, and discloses the information. This Privacy Policy applies to all of our users, including children under the age of 13. The Mamababa mobile application values the privacy of children and all of its users. Our Privacy Policy is designed to comply with the Children’s Online Privacy Protection Act(COPPA).

Personal Information and Use

The Mamababa app does not have a registration process and personal information is not collected. The gender and birthdate data that users enter is used only for the purpose of generating video playlists and any user can change the data anytime. The gender data does not affect the video playlist in any way, and when the birthdate changes to a different age the video playlist will change accordingly. The app does not use third party services in any way; therefore, data related to users is not shared with any other party.

Data Collection

When the Mamababa app is downloaded, a “User ID” is automatically generated for each user and it consists of a combination of random alphabets. The data collected through the User ID is used for the following purposes.

  1. To identify the usage time for all of our users
  2. To identify the category preferences for all of our users
  3. To identify the number of users in each age group

The data collected through the User ID will only be used for contents analysis by Gene Lee.
